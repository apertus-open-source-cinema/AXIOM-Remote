AXIOM Remote 3D Viewer

This enclosure concept represents the current state of the design of the open hardware/free software AXIOM Remote with all buttons, knob and LCD. The original firmware running on the device is also running on this 3D mockup and you can interact with buttons, etc. like you would with the real device.

Usage:
Left Mouse Button - interact with knob/buttons
Right Mouse Button - Rotate View
Middle Mouse Button - Pan View
Mouse Wheel - Zoom View
